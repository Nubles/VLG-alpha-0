# Repository Instructions

- This is a custom C++ game engine project for Realmbound Wilds.
- Do not suggest Unity, Unreal Engine, Godot, or another existing commercial/game engine.
- Keep `Engine` and `Game` code separate.
- Every change must compile or clearly explain what remains incomplete.
- Prefer small, reviewable commits or patches.
- Before coding, inspect existing files.
- After coding, list changed files and testing steps.
- Do not create broad systems before their roadmap milestone.
- Use placeholder assets and primitive debug rendering first.
- Optimize for a solo developer.
- Avoid copying Valheim's protected expression, names, UI, lore, art, enemies, bosses, or exact mechanics.
